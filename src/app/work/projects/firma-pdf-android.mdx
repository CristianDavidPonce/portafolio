## Overview

Aplicación móvil Android nativa que permite firmar documentos PDF utilizando certificados digitales p12/pfx de manera completamente offline, garantizando la privacidad y seguridad de tus documentos.

## Key Features

- **Firma Digital Certificada**: Soporte completo para certificados p12/pfx
- **Compatible con Firma Electrónica Ecuador**: Funciona con certificados emitidos en Ecuador 🇪🇨
- **100% Offline**: No requiere conexión a internet, tus documentos nunca salen de tu dispositivo
- **Rápido y Seguro**: Procesamiento local que garantiza la privacidad de tus datos
- **Interfaz Intuitiva**: Diseño simple y fácil de usar para firmar documentos en segundos
- **Nativo Android**: Desarrollado con Kotlin para máximo rendimiento

## Technologies Used

- **Language**: Kotlin
- **Platform**: Android
- **PDF Processing**: Librerías nativas de manipulación de PDF
- **Security**: Manejo seguro de certificados digitales

## Challenges and Learnings

El principal desafío fue implementar un sistema robusto de procesamiento de certificados digitales y firma de PDFs que funcionara completamente offline. Fue necesario investigar en profundidad los estándares de firma digital y asegurar la compatibilidad con los certificados oficiales de Ecuador.

## Outcome

La aplicación está disponible en Android y está siendo utilizada para firmar documentos de manera segura y privada, sin necesidad de subir archivos sensibles a servicios en la nube, garantizando total control sobre la información del usuario.
