## Overview

Blog personal desarrollado con stack completo, implementando autenticación robusta y sistema de autorización basado en roles y permisos.

## Key Features

- **Backend Robusto**: Desarrollado con NestJS, implementando patrones de diseño y arquitectura limpia
- **Autenticación JWT**: Sistema seguro de autenticación usando JSON Web Tokens
- **Autorización Basada en Roles**: Control de acceso granular con sistema de roles y permisos
- **Base de Datos MySQL**: Gestión eficiente de datos con TypeORM como ORM
- **Frontend Moderno**: Interfaz desarrollada con Next.js y Tailwind CSS
- **Editor Avanzado**: Integración de EditorJS para creación de contenido rico

## Technologies Used

- **Backend**: NestJS, MySQL, TypeORM, JWT
- **Frontend**: Next.js, React, Tailwind CSS
- **Editor**: EditorJS

## Challenges and Learnings

La implementación de un sistema de autorización basado en roles y permisos requirió un diseño cuidadoso de la arquitectura de backend. El desafío principal fue balancear la seguridad con la experiencia de usuario, logrando un sistema que es tanto robusto como fácil de usar.

## Outcome

El blog está actualmente en producción y funcionando de manera estable, permitiendo la creación y gestión de contenido de forma eficiente y segura.
