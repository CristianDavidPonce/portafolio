## Overview

Sistema completo para la administración de centros médicos y hospitales, diseñado para optimizar la gestión operativa y mejorar la atención al paciente.

## Key Features

- **Gestión de Pacientes**: Módulo completo para registro y seguimiento de pacientes
- **Sistema de Agenda**: Programación y gestión de citas médicas
- **Módulo de Imagen**: Gestión de estudios de imagen y diagnósticos
- **Farmacia**: Control de inventario y dispensación de medicamentos
- **Terapia**: Seguimiento de tratamientos terapéuticos
- **Inventario**: Control integral de recursos y suministros médicos
- **Usuarios y Roles**: Sistema de permisos y accesos basado en roles

## Technologies Used

- **Frontend**: TypeScript, React, Next.js, Ant Design
- **Backend**: Node.js, TypeScript
- **Database**: MongoDB
- **UI Framework**: Ant Design

## Challenges and Learnings

El principal desafío fue diseñar un sistema que pudiera manejar múltiples módulos interconectados mientras mantenía la simplicidad en la interfaz de usuario. La integración de todos los módulos requirió una arquitectura escalable y mantenible.

## Outcome

El sistema está siendo utilizado activamente en centros médicos, mejorando significativamente la eficiencia operativa y la calidad de la atención al paciente.
