## Overview

Aplicación móvil multiplataforma diseñada para que clientes de estudios jurídicos puedan hacer seguimiento de sus casos, trámites y acceder a herramientas de cálculo laboral.

## Key Features

- **Seguimiento de Trámites**: Visualización en tiempo real del estado de trámites
- **Gestión de Casos**: Acceso a información detallada de casos legales
- **Calculadora de Jubilación Patronal**: Herramienta para calcular prestaciones
- **Calculadora de Liquidaciones**: Cálculo de liquidaciones laborales
- **Notificaciones**: Alertas sobre actualizaciones de casos
- **Material Design 3**: Interfaz moderna y accesible

## Technologies Used

- **Framework**: React Native con Expo
- **Language**: TypeScript
- **UI**: Material Design 3
- **Platforms**: iOS, Android

## Challenges and Learnings

El desafío principal fue desarrollar calculadoras precisas que cumplieran con la legislación laboral vigente, además de crear una interfaz que fuera fácil de usar para usuarios no técnicos. Se trabajó en estrecha colaboración con expertos legales para validar los cálculos.

## Outcome

La aplicación está disponible en ambas plataformas móviles y ha facilitado significativamente la comunicación entre abogados y clientes, además de proporcionar herramientas útiles de cálculo que anteriormente requerían consultas presenciales.
