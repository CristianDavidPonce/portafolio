## Overview

Visor de imágenes médicas DICOM completamente basado en web, diseñado para facilitar el análisis y diagnóstico por imagen sin necesidad de software de escritorio.

## Key Features

- **Visualización DICOM**: Soporte completo para el formato estándar de imágenes médicas
- **Herramientas de Medición**: Reglas y medición de ángulos para análisis preciso
- **Ajustes de Imagen**: Control de brillo, contraste y otros parámetros visuales
- **Zoom Avanzado**: Funcionalidad de acercamiento para inspección detallada
- **Interfaz Intuitiva**: Diseño enfocado en la experiencia del usuario médico
- **100% Web**: Sin necesidad de instalación de software adicional

## Technologies Used

- **Frontend**: TypeScript, React
- **DICOM**: Procesamiento de imágenes médicas en formato DCM
- **Canvas API**: Renderizado eficiente de imágenes

## Challenges and Learnings

El principal desafío fue la optimización del rendimiento al manejar archivos DICOM de gran tamaño en el navegador. Se implementaron técnicas de carga progresiva y renderizado eficiente para asegurar una experiencia fluida.

## Outcome

La herramienta está siendo utilizada por profesionales médicos, permitiendo el análisis de imágenes médicas de manera rápida y eficiente desde cualquier navegador web.
