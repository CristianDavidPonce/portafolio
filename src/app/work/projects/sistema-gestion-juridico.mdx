## Overview

Sistema de gestión completo diseñado específicamente para estudios jurídicos, facilitando la administración de clientes, casos legales y trámites de manera eficiente.

## Key Features

- **Gestión de Clientes**: Base de datos completa de clientes con toda su información relevante
- **Administración de Trámites**: Seguimiento de procesos y documentación legal
- **Gestión de Casos**: Control detallado de casos legales en curso
- **Sistema de Usuarios y Roles**: Control de acceso basado en permisos
- **Interfaz Profesional**: Diseño limpio y profesional con Ant Design
- **Reportes**: Generación de informes y documentos legales

## Technologies Used

- **Frontend**: TypeScript, React, Next.js, Ant Design
- **Backend**: Node.js, TypeScript
- **Database**: MongoDB

## Challenges and Learnings

El principal desafío fue comprender los flujos de trabajo específicos del ámbito legal y traducirlos en funcionalidades de software útiles. La colaboración cercana con abogados fue clave para crear un sistema que realmente se ajustara a sus necesidades.

## Outcome

El sistema está siendo utilizado activamente por estudios jurídicos, mejorando significativamente la organización y eficiencia en la gestión de casos y clientes.
